# BioShock: Manual de Supervivencia

## Descripción
Este proyecto es una página web desarrollada en HTML como parte del Proyecto Integrador de Programación III. 
La página presenta una guía de supervivencia en la ciudad de Rapture del videojuego BioShock.

## Contenido
- Historia de Rapture
- Tecnología (Plásmidos y Tonics)
- Guía de supervivencia
- Tabla de plásmidos
- Imágenes y video
- Formulario de contacto

## Tecnologías utilizadas
- HTML5
